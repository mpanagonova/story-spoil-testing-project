# Story Spoil QA Testing Project

This repository contains manual and API testing documentation created during the QA Fundamentals and Manual Testing training.

Note:
The original testing environment used for this training project
is no longer available online. This repository presents the
test documentation, bug reports and API testing collection
created during the testing process.

## Application Under Test
Story Spoil Web Application

The application allows users to:
- register accounts
- log in
- create spoilers
- edit profiles
- manage spoilers

## Testing Activities

Manual Testing
- test cases for key user scenarios
- defect reporting
- validation of functional requirements

API Testing
- REST API testing with Postman
- authentication with access token
- CRUD operations for spoilers

## Test Coverage

The following use cases were tested:

1. Home Page access
2. User Registration
3. User Login
4. Profile Management
5. Spoiler Creation
6. Spoiler Management
